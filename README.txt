ML Rutas Optimizer

Cómo usar:
1. Abre index.html en el navegador.
2. Ve a la pestaña Imágenes y sube capturas.
3. Pulsa Leer capturas.
4. Si el OCR no carga, publícalo en GitHub Pages o ábrelo con un servidor local.

Nota importante:
- El OCR usa Tesseract.js y necesita cargar su motor.
- Si abres el archivo como file:// y tu navegador bloquea el motor OCR, usa GitHub Pages o un servidor local.
- Si tienes Python instalado, puedes abrir una terminal en esta carpeta y correr:
  python -m http.server 8080
  Luego abre: http://localhost:8080
